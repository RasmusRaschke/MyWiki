``` ad-Definition
title: Definition (Free Action).

Let $G$ be a group acting smoothly on a smooth manifold $M$. The action is __free__ if for all $g \in G$ with $g \neq e$ the diffeomorphism $$\phi_g: M \to M$$ has no [[fixed points]].
```
