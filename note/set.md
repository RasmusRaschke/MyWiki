# Partition

``` ad-Definition
title: Definition (Partition).

Let $S$ be a set. A <ins>partition</ins> of $S$ is a family $\sP$ of disjoint non-empty subsets of $S$ such that $$\bigcup_{A \in \sP} A = S.$$ 
```
